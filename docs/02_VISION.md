# 02 — Vision

## Future Ideal State

Every Leadership Team role in BNI Chapter Mahardika has a dedicated AI assistant
and web dashboard. LT members focus on strategic decisions. All data flow,
administration, reporting, and routine communication is handled by AI.

## Current State (April 2026)

Manual Google Forms → data scattered
Excel exports → disconnected
WhatsApp groups → information buried
BNI classifications → inaccurate
Power Teams → formed by assumption
Referrals → frequently misdirected
Reports → compiled by hand, late
Handover → knowledge lost every term

## Target State

Member joins chapter
  → Bot auto-sends welcome + business profile form
  → AI parses profile → assigns to Power Team
  → Weekly: attendance 1-tap, referral prompt, "Member You Should Meet"
  → Monthly: dashboard refresh, health score, gap analysis
  → Visitor arrives: QR scan → auto-match to wanted list → follow-up sequence
  → Event: create via bot → publish all channels → registration → report

## Automation Maturity Levels

Level 1 — DIGITIZE (Phase 1, now)
  CB starts: data via form & upload, stored in database, accessible via login.

Level 2 — ASSIST (Phase 2-3)
  CB + VP: dashboard + bot active. AI parses data, generates insights.

Level 3 — AUTOMATE ALL ROLES (Phase 4-5)
  Every LT role has: dedicated dashboard, AI assistant (alert, suggest, draft),
  bot commands per role, auto-generated reports.

Level 4 — INTELLIGENCE (Phase 6)
  Cross-role AI: chapter health composite, churn prediction, growth forecast,
  prescriptive recommendations.

Level 5 — SCALE (Future)
  Template for other chapters. Multi-chapter dashboard. Inter-chapter referrals.
